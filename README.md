# 👤 About Me
👋 Olá! Sou o Turgho, desenvolvedor backend com experiência prática em projetos reais, focado em construir **APIs e sistemas distribuídos** eficientes e bem estruturados.

🧑‍💻 Minha stack principal é **Go** e **C#/.NET**, com vivência em filas de mensageria (RabbitMQ), múltiplos bancos de dados e contêineres (Docker).

🌱 Aplico boas práticas como **SOLID**, **Clean Architecture** e **TDD**, e também desenvolvo frontends em **Angular** para complementar os projetos full-stack.

🚀 Estou cursando Análise e Desenvolvimento de Sistemas (Cruzeiro do Sul) e em transição de carreira para atuar formalmente como dev.

---

## 🔭 Projetos em Destaque:

- **[Aluguei](https://github.com/Turgho/Aluguei)** ★ ativo — PWA completo de gestão de imóveis: Go (Gin + GORM), autenticação JWT em cookies HttpOnly, hash Argon2id, Angular + TailwindCSS, Docker.
- **[go-streaming-plataform](https://github.com/Turgho/go-streaming-plataform)** ★ ativo — Monorepo de microsserviços em Go (gRPC, NATS, MongoDB) com transcodificação HLS via ffmpeg e auth JWT.
- **[Shinobu-Whatsapp](https://github.com/Turgho/Shinobu-Whatsapp)** — Bot de WhatsApp em Go com IA contextual (Groq/LLaMA), busca web (Tavily), efeitos de áudio (ffmpeg) e música (yt-dlp).
- **[BarberFast](https://github.com/Turgho/BarberFast)** — Sistema de agendamento para barbearias: Go (Gin + GORM), RabbitMQ, JWT, Angular.
